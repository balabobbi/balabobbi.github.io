SEM EDT Electronic Diagnostic Tool 2021B is a powerful diagnostic tool used to communicate with Electronic Control Modules (ECMs) via a data link. It allows users to diagnose existing or potential issues, configure products, and retrieve data for analysis. With this tool, you can view active and logged diagnostics, monitor parameter statuses like temperatures and pressures, record performance data, graph status parameters, change ECM configurations, perform diagnostic tests and calibrations, and print detailed reports. It is an essential tool for efficient and accurate electronic system management.

It allows:

  • View active and logged diagnostics
  • View events where irregularities occurred and were logged by the ECM
  • View the status of a group of parameters (temperatures, pressures, etc.) simultaneously
  • Record and log performance data
  • Graph a group of status parameters
  • View the current configuration of an ECM
  • Change ECM configurations
