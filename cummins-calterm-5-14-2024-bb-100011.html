Cummins Calterm 5.14 — this is a tool used to monitor Cummins electronic control modules and change the calibration data they contain. Designed for changing fuel calibration parameters and feature settings in an engineering development and testing environment.
The program works in conjunction with the interface of the RP1210 standard (for example, Cummins Inline, Nexiq or CarDaq)

Description:

  • diagnose ECU,
  • remove ECU password protection,
  • adapt, program, change ECU configuration,
  • disable ADBlue, DPF,
  • increase power,
  • remove speed limit,
  • etc.
