ISTA-D is used for diagnosing all BMW series and for coding F, G, and I series. For the E series, ISTA-P should be used.

After identifying faults, the program allows for the creation of a step-by-step plan for troubleshooting, including specific electrical diagrams of the location if there is a break. The program also allows for forced activation of any mechanically controlled units with electronic management, such as starting the wipers, directing the beam of adaptive headlights in any specified direction, fully raising/lowering the air suspension, setting an individual road height, or adjusting corrections in millimeters for the right and left sides. There is an option to re-adapt the rain sensor if there is discomfort from the wipers turning on too infrequently.

New versions work only on Windows 10 x64, but if you install the 2020 version, it can also work on Windows 7 x64.
