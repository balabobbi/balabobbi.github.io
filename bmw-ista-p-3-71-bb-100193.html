BMW ISTA-P 3.71 is the programming software used for BMW, MINI, and Rolls-Royce vehicles. Part of the Integrated Service Technical Application (ISTA) system, it replaces older programs, offering advanced diagnostics and reprogramming of electronic control units (ECUs). ISTA-P 3.71 provides crucial data for vehicle programming, coding, and updates, ensuring compatibility with future features and improvements in the vehicle's software and systems.

It works with E-series. For Coding / Programming on E-series Cars.

Previous versions support E, F, G, and I series. Currently, the remaining versions are in ISTA-D.

The functionality allows you to:

  • Read error codes
  • Read data stream (LiveData, engineering parameters)
  • Erase fault codes
  • Program (for all vehicles operating without the use of an optical data bus)
  • Reset service lights and intervals
  • Check real-time data
  • Perform DME/DDE synchronization
  • Perform EWS/CAS synchronization
  • Adjust the air-fuel mixture
  • Conduct tests (activation of actuators)
  • Code (for all vehicles operating without the use of an optical data bus)
  • Reset airbag lights
  • Program new components, with an option for updates
  • Check wheel speed, ABS
  • Perform full diagnostics (including test plans)
  • Create BMW-approved repair, update, and upgrade test plans
  • Monitor live data streams with graph display and information on the vehicle's version
  • Reset adaptation settings
  • Reset ECU adaptations
  • Register new batteries after replacement
  • Update software and firmware
  • Code ECU (seat memory in the key, daytime running lights, sound signal after locking)
  • And many other functions. The software also includes wiring diagrams and spare parts catalogs. It has everything you would find in BMW dealership centers.
