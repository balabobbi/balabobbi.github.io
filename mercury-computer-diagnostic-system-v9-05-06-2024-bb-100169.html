The Mercury Computer Diagnostic System Version 9.05 (June 2024) is a comprehensive diagnostic tool designed for marine and racing applications. It supports a range of machines including:

  • MARINER MERCRUISER: Systems for various Mariner and Mercruiser engines.
  • RACE OUTBOARD: Diagnostic capabilities for high-performance racing outboard engines.
  • M2 JET DRIVE 1: Specific diagnostics for the M2 Jet Drive 1 system.
  • SPORT JE: Support for Sport JE engines, focusing on high-performance and sport models.
  • GENERIC RACE: General diagnostics for various racing engines.
  • STERNDRIVE: Diagnostics for Mercruiser stern drives, offering in-depth analysis and troubleshooting.
  • CUMMINS MERCRUISER: Integration with Cummins-powered Mercruiser systems.

This version includes updates and enhancements to improve diagnostic accuracy and support a wide range of marine and racing engines.

Mercury Marine and Cummins MerCruiser Diesel are proud to now offer diagnostic support for the new QSD series diesel engines with SmartCraft capability. Simply select the Cummins MerCruiser Diesel Engine on the Engine Selection screen to access this new portion of the software. For more information on cables needed and basic functionality, please review the Mercury Computer Diagnostic System Help File.

What's New:

  • Version 9.05 Mercury Computer Diagnostic Software (CDS) contains the following features.
  • Support for Windows XP and Windows 7 (32 and 64 bit) operating systems.
  • Support for reflash of Verado PCM08 processors.
  • Support for the use of a USB to Serial adapter for the SmartComms box, Multimeter and Pressure Tester. Com 1 port is no longer required on PC’s running MCDS.
  • Adobe reader version 10.0.1.
